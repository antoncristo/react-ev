# Checkbox

<h3>props</h3>
<ul>
	<li>
		<b>value</b> - input's value to read on the onChange handler
	</li>
	<li>
		<b>isChecked</b> - the state of the checkbox (show/hide marker)
	</li>
	<li>
		<b>onCheckboxSelect</b> - a handler to alter the isChecked state
	</li>
	<li>
		<b>inputID</b> - an id, to sync with a label tag, if needed
	</li>
	<li>
		<b>width</b> - set the size of the checkbox square
	</li>
	<li>
		<b>colors</b> - update the colors separately
	</li>
	<li>
		<b>html attributes</b> - standard html button attributes
	</li>
</ul>
