# Draggable

<h3>props</h3>
<ul>
	<li>
		<b>children</b> - the content that would be draggable
	</li>
	<li>
		<b>onDragStart/onDragOver/onDrop</b> - DragEventHandler (see draggable API)
	</li>
	<li>
		<b>value</b> - (optional) a string to be used as the data-value attribute on the drag
		events
	</li>
	<li>
		<b>index</b> - (optional) a string to be used as the data-index attribute on the drag
		events
	</li>
</ul>
