# Table

<h3>props</h3>
<ul>
	<li>
		<b>data</b> - the data array of any object type
	</li>
	<li>
		<b>columns</b> - a configuration array of columns (see Column type)
	</li>
	<li>
		<b>sortSetup</b> - (optional) if sort needed, sort configuration object is required
	</li>
	<li>
		<b>selectSetup</b> - (optional) if row selection functionality needed, define this
		handler
	</li>
</ul>
