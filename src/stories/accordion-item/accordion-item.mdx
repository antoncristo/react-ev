# AccordionItem

<h3>props</h3>
<ul>
	<li>
		<b>children</b> - any valid JSX component
	</li>
	<li>
		<b>accordionKey</b> - the clickable key item (collapsing functionality)
	</li>
	<li>
		<b>isCollapsed</b> - (optional) if passed, then the 'setIsCollapsed' is required to
		have a controlled state
	</li>
	<li>
		<b>setIsCollapsed</b> - (optional) if passed, then the 'isCollapsed' state is required
		to have a controlled state
	</li>
	<li>
		<b>chevronColor</b> - (optional) set the color of the chevron icon
	</li>
	<li>
		<b>html attributes</b> - standard html attributes
	</li>
</ul>
