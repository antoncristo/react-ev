# Dropdown

<body>
	<h3>props</h3>
	<ul>
		<li>
			<b>context</b> - the component that provides the dropdown
		</li>
		<li>
			<b>header</b> - an optional header component for the dropdown
		</li>
		<li>
			<b>footer</b> - an optional footer component for the dropdown
		</li>
		<li>
			<b>listConfig</b> - configuration object to handle the inner list of the dropdown
			(List component props)
		</li>
		<li>
			<b>position</b> - absolute position values (top,left,bottom,right)
		</li>
		<li>
			<b>html attributes</b> - standard html attributes
		</li>
	</ul>
</body>
