# Alert

<h3>props</h3>
<ul>
	<li>
		<b>isOpen</b> - represents the visibility state
	</li>
	<li>
		<b>setIsOpen</b> - a boolean toggler that control the visibility state
	</li>
	<li>
		<b>variant</b> - alert coloring - success\error\warning\default (depends on theme base
		colors)
	</li>
	<li>
		<b>text</b> - alert's text, the message that is being rendered
	</li>
	<li>
		<b>title</b> - (optional) the title of the popup
	</li>
	<li>
		<b>textConfig</b> - (optional) inner text props, allows a modification of the message
		text style (see TextField props)
	</li>
	<li>
		<b>backdropColor</b> - (optional) set the color of the viewport background
	</li>
</ul>
