# TextField

<h3>props</h3>
<ul>
	<li>
		<b>children</b> - a text in string format only
	</li>
	<li>
		<b>fontSize</b> - font size as string, can be set in any unit ( defaults to 1rem )
	</li>
	<li>
		<b>fontWeight</b> - font weight as string, can be set in any unit - text or numeric (
		defaults to 'normal' )
	</li>
	<li>
		<b>lineHeight</b> - line height as string, can be set in any unit ( defaults to 1.2rem
		)
	</li>
	<li>
		<b>maxWidth</b> - max width of the text field
	</li>
	<li>
		<b>numberOfLines</b> - max number of lines. If text overflows than an ellipsis is set
		on the last line
	</li>
	<li>
		<b>html attributes</b> - standard html attributes
	</li>
</ul>
