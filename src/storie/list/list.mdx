# List

<h3>props</h3>
<ul>
	<li>
		<b>list</b> - a generic list of the data (MUST extend object type)
	</li>
	<li>
		<b>itemKey</b> - a key of the rendered data object
	</li>
	<li>
		<b>renderItem</b> - a render function that receives a single object from the list and
		returns a JSX list item element
	</li>
	<li>
		<b>gap</b> - the space each list item in pixels ( defaults to 0 )
	</li>
	<li>
		<b>height</b> - the height of the list ( defaults to 'fit-content' )
	</li>
	<li>
		<b>html attributes</b> - standard html attributes
	</li>
</ul>
